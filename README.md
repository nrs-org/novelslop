# novelslop

The _Hokori Takaki Idol_ novel, now on Git!

_"novelslop" is a self-depreciating name I used to refer to this work, therefore
the repo name._

Hop on AO3, this is mainly a place for me to write. There is some extra content
here, but not much.

**AI usage disclaimer**: I come from a CS/IT background where AI (more
specifically, transformer-based LLMs like ChatGPT) usage is prevalent, so this
novel is in no way AI-free. However, AI usage is limited to only very basic
stuff (that a search engine can mostly do):
- Synonym/Opposite words searching
- Paraphrasing a (very) few (say less than 1%) phrases that I could not really
  put the words.
- Worldbuilding, I guess? Like I asked it about some general fact about Japan? I
  don't live there, yeah.

(This does not apply to any CODE in this repo, though. I am still using AI to
automate things because my free time is limited. For artists, in the coding
sphere, AI-assisted work is pretty much normalized nowadays, for good reasons,
IMO.)

Since this novel is going public, AI usage will be reduced in place of less
expensive and user-invasive technology. Personally, I think while this novel is
not 100% AI-free, more than 99% of it can be considered this way.

I'm welcomed to issues/PRs (but I think you should just post a comment on AO3
instead of GitHub issues IMO), but I don't really see any content I can merge
(Illustrations? Too high effort to give me for free though). Please avoid AI if
you are submiting patches, too.
